# <a name="UniqueLanguage"><h1>Unique language</h1></a>

The main idea use language in dialog, negotiations, conferences. not only in code and documentation. Use a model as a base for build language.  

The component of effective buld model:
* create a relationship with model end realization
* use a language that a basic on the model
* create a detail model schema
* experiments and use a brainstorm

Refining knowledge: You need to be prepared for the fact that knowledge will change. And they are not afraid to work on the processing of knowledge. Add the new world and change exist world in the model and in the code too. You can upgrade language describe voice model that you build sit on the conference and write most uses words.
